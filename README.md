# Agente Financeiro Inteligente (Protótipo)
Objetivo

# Desenvolver um protótipo de agente financeiro capaz de:

Antecipar necessidades do usuário
Personalizar sugestões financeiras
Auxiliar na tomada de decisões
Garantir respostas seguras e confiáveis

# Tecnologias Utilizadas
Python
Lógica de programação
Estruturas de dados (dicionários)

# Funcionamento do Sistema

O agente coleta informações do usuário, como:

Nome
Saldo atual
Gastos mensais
Objetivo financeiro

Com base nesses dados, ele realiza uma análise e fornece recomendações personalizadas.
